Задача 01. Старт
=
Ограничение времени: 0.5 сек. Ограничение памяти: 64 Мб

В гонке с раздельным стартом спортсмены стартуют по очереди, с интервалом в 30 секунд. В такой гонке участник, финишировавший первым, не обязательно будет первым в итоговой таблице результатов. Требуется написать программу, определяющую спортсменов, результат которых был лучшим на момент их финиша.

## Исходные данные

В первой строке записано целое число N - количество участников гонки (1 ≤ N ≤ 100). В i-й из следующих N строк через пробел записаны имя спортсмена, стартовавшего i-м, и время, за которое он прошёл трассу, в формате mm:ss.d, заданное с точностью до десятых секунды. Гарантируется, что никакие два спортсмена не финишировали одновременно и никакие два спортсмена не показали одинаковое время. Имя спортсмена - непустая строка, состоящая из английских букв, длиной не более 20 символов. Имена всех спортсменов различны.

## Результат
В первой строке выведите количество спортсменов, которые были лидерами гонки на момент их финиша. Далее выведите имена этих спортсменов в лексикографическом порядке, по однму в строке.

## Пример
######Ввод:
6  
Valera 21:38.2  
Anatoly 21:21.0  
Mikhail 22:04.4  
John 22:06.1  
Boris 21:11.1  
Oleg 22:05.8  
######Вывод:
3  
Anatoly  
Boris  
Valera

Задача 04. Шары
=
Ограничение времени: 1.0 сек. Ограничение памяти: 64 Мб

Имеется N последовательно пронумерованных числами от 1 до N шаров, каждый из которых либо белого, либо черного цвета. Также имеется K аналогично пронумерованных числами от 1 до K ящиков.

Шары можно помещать в ящики следующим образом: первые N1 шаров (в соответствии с порядком их нумерации) помещаются в ящик с номером 1, следующие N2 шаров - в ящик с номером 2, и т.д.
При этом вариативностью ящика будем называть величину w*b, где w - количество белых, а b - количество черных шаров, помещенных в этот ящик. Общей вариативностью будем называть сумму вариативностей всех K ящиков.

Необходимо разложить все N шаров по K ящикам описанным способом так, чтобы ни один ящик не остался пустым, и общая вариативность была минимальной.

## Исходные данные
В первой строке заданы два целых числа: N (1 ≤ N ≤ 500) и K (1 ≤ K ≤ N). В следующих N строках заданы N чисел. i-я из этих строк определяет цвет i-го шара: 0 - белый, 1 - черный.
## Результат
Необходимо вывести единственное число, которое будет являться минимально возможным значением общей вариативности.
## Пример
######Ввод
6 3  
1  
1  
0  
1  
0  
1  
######Вывод 
2  

## Пояснение к примеру
Поместить первые 2 шара в первый ящик, следующие 3 шара во второй ящик, и последний шар в третий ящик.

Задача 06. Сдвиг
=
Ограничение времени: 1.0 сек. Ограничение памяти: 64 Мб
Даны две строки S и T из N символов каждая. Необходимо определить, может ли строка T быть получена из строки S некоторым количеством X (0 ≤ X < N) последовательных циклических сдвигов. Под циклическим сдвигом строки понимается перенос её последнего символа в начало.
Исходные данные:
Первая строка содержит целое число N (1 ≤ N ≤ 250000). Вторая строка содержит строку S. Третья строка содержит строку T. Обе строки имеют длину N и могут содержать любые символы таблицы ASCII с кодами от 33 до 255.
Результат:
Если строка T может быть получена из строки S с помощью циклических сдвигов, выведите искомое число X, иначе выведите "-1". Если задача имеет несколько решений, можно вывести любое из них.
Пример:
Ввод:
11
abracadabra
acadabraabr
Вывод:
8

Задача 07. Набор
=
Ограничение времени: 0.5 сек. Ограничение памяти: 64 Мб

Порождающим набором назовем некоторый набор различных целых положительных чисел. Просуммировав все цифры чисел порождающего набора, получим некоторое число N. Например, после обработки набора чисел 7, 12, 43 получим число N = 17 = 7 + 1 + 2 + 4 + 3.

Необходимо построить порождающий набор для заданного числа N, либо определить, что такого набора не существует.

##Исходные данные
В единственной строке записано целое число N (0 < N < 105).
##Результат
Если для данного N можно построить порождающий набор, в первой строке выведите количество чисел в таком наборе, а во второй строке выведите через пробел сами эти числа. Все числа должны быть целыми, положительными и строго меньшими 105. Кроме того, все числа в наборе должны быть различными. Если существует несколько порождающих наборов, выведите любой из них. Если ни одного порождающего набора не существует, выведите -1.

##Пример
######Ввод
17
######Вывод
3  
7 12 43

Задача 08. Этажи
=
Ограничение времени: 0.0 сек. Ограничение памяти: 0 Мб

Существует здание, имеющее (N+1) этажей, пронумерованных от 0 до N. На каждом этаже здания расположено некоторое количество комнат.

В каждой комнате i-го этажа есть несколько переходов, позволяющих подняться в некоторые комнаты (i+1)-го этажа (спускаться вниз по этим переходам нельзя). Каждый переход имеет некоторую стоимость, выраженную целым числом. Стоимость может иметь и отрицательное значение.
Необходимо найти способ добраться с 0-го этажа до какой-нибудь комнаты N-го этажа за минимальную совокупную стоимость. Известно, что хотя бы один путь с 0-го этажа в одну из комнат N-го этажа существует.
Исходные данные:
В первой строке указано число N (0 < N < 30) - номер последнего этажа здания. Далее следует N блоков информации, описывающих переходы между этажами. А именно, i-й блок описывает схему переходов из комнат (i-1)-го этажа в комнаты i-го этажа. Блоки разделяются строками, содержащими один символ "*". Комнаты на каждом этаже нумеруются последовательными целыми положительными числами от единицы. На каждом этаже не более 30 комнат. На нулевом этаже только одна комната - та, с которой начинается путь. В первой строке блока записано число Ki - количество комнат на i-м этаже. Далее следуют Ki строк - по одной для каждой комнаты i-го этажа. В каждой такой строке перечислены через пробел номера комнат предыдущего, (i-1)-го этажа, с которых можно перейти в данную комнату, и стоимости таких переходов. Стоимость каждого перехода - целое число от -32768 до 32767. Каждая строка с описанием переходов в комнату завершается нулем.
Результат:
Выведите единственное целое число - минимальную стоимость перехода в какую-нибудь комнату N-го этажа. Ответ может быть и отрицательным.
Пример:
Ввод:
3
2
1 15 0
1 5 0
*
3
1 -5 2 10 0
1 3 0
2 40 0
*
2
1 1 2 5 3 -5 0
2 -19 3 -20 0
Вывод:
-1

Выведите единственную строку, содержащую расшифрованный текст. Строка должна оканчиваться символом перевода строки.

##Пример
